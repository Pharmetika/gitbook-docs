# Installing BrowserPrint for Zebra Printers

{% hint style="info" %}
Browser Print supports the following Zebra printers:

* QLn series
* ZT200™ series
* ZT400™ series
* ZT510
* ZT600 Series
* ZD500™ series
* ZD400™ series
* LP2824+
* GK420
* GX420
{% endhint %}
