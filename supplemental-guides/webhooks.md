# Webhooks

This webhook configuration option is available in the **Provider Portal**, offering UI support for an already existing functionality. This feature enables providers to set up webhooks to receive real-time updates and notifications from the system. The configuration can be accessed under the portalâ€™s settings, where users can define webhook endpoints and manage their integration preferences.

To add a new webhook in the Provider Portal, navigate to the **Configuration** > **Webhook Configuration** >  **Add New Webhook Registration**.
